# Wai Lan Garden 商店管理系统 - UI 需求文档

**版本：** 1.0  
**日期：** 2026-03-07  
**项目：** IIMT4601 Track II - Flowers App

---

## 📋 系统概述

### 目标用户
- **花店店主/员工** - 管理订单、库存、花卉目录
- **非技术背景** - 界面需简单直观，操作便捷

### 核心功能
1. 订单管理 - 查看、处理、追踪订单
2. 库存管理 - 实时更新花卉库存
3. 花卉目录管理 - 添加/编辑/下架花卉
4. 数据统计 - 销售报表、热门商品

## 🔧 技术实现建议

### 前端框架选择
| 选项 | 优点 | 缺点 |
|------|------|------|
| SwiftUI (iPad) | 与现有 App 一致 | 仅支持 Apple 设备 |
| React Web | 跨平台，易部署 | 需额外学习 |
| Flutter | 跨平台，性能好 | 需额外学习 |

**推荐：** 如果店主用 iPad 管理 → **SwiftUI**；如果用电脑 → **React Web**

### 数据同步
- 使用 Firebase Firestore 实时同步
- 离线缓存支持
- 冲突解决机制 (最后写入优先)

### 权限控制
```javascript
// Firestore 安全规则示例
match /orders/{orderId} {
  // 仅管理员可读写订单
  allow read, write: if request.auth != null && 
                       get(/databases/$(database)/documents/admins/$(request.auth.uid)).data.role == 'admin';
}
```
